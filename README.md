# Cogie Auth Login Screen UI

🌍 **Live Demo:** [Check it out here!](https://cogieauthui.netlify.app/)

A sleek, responsive, and highly interactive login screen developed with pure HTML, CSS, and Vanilla JavaScript. Designed with modern aesthetics in mind, this project demonstrates how to build a premium user interface without relying on heavy frontend frameworks.

## ✨ Features

- **Dark/Light Mode:** Seamlessly toggle between dark and light themes. The system automatically detects your OS preference on load.
- **Dynamic Animations:** Staggered fade-ins, slide-ups, and smooth hover transitions make the interface feel alive.
- **Interactive Floating Labels:** Input placeholders smoothly transition into floating labels upon focus.
- **Real-Time Password Strength Meter:** Visual feedback bar evaluating password complexity (length, capitalization, numbers, special characters) as you type.
- **Caps-Lock Warning:** Intelligently detects and warns you if Caps-Lock is on while typing in the password field.
- **Fully Responsive:** Adapts elegantly from desktop to mobile screens, stacking the split-panel design into a scrollable card on smaller devices.
- **Zero Dependencies:** Built entirely with vanilla web technologies. No `npm install`, Node.js, or build steps required.

## 🚀 Getting Started

1. Clone or download this repository.
2. Ensure you have the background image named `bg_image.jpg` placed in the root folder alongside the HTML file.
3. Simply open `index.html` in your favorite web browser to view the project.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure and accessible ARIA labels.
- **CSS3:** Custom variables (for theme switching), Flexbox, Glassmorphism ambient glow, and Keyframe animations.
- **JavaScript (ES6):** DOM manipulation, event listeners for real-time validation, and theme state management.
- **Fonts:** [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) (headings) and [Inter](https://fonts.google.com/specimen/Inter) (body) from Google Fonts.
